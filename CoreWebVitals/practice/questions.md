**LCP**

- What is LCP?
    - LCP stands for Largest Contentful paint and its a metiric that measures how fast the biggest element in the page was loaded and visible to the user.

- What does it measure on the page?
    - It measures how fast the biggest element in the page was able to appear visible to the user.

- When is it considered “good”?
    - ≤ 2.5 seconds = ✅ Good (user sees main content quickly)

**CLS**

- What is CLS?
    - CLS is cummulative Layout shift.

- What kind of behavior does it measure?
    - It measures how much layout shift occurs when interacting with the page.

- What does a high CLS indicate?
    - A high CLS means the page has lots of unexpected layout shifts—content jumps around while loading or interacting. This creates a poor, frustrating user experience because things move when users don't expect it (e.g., buttons move, text shifts, ads push content down).

**INP** (or interaction metric)

- What is INP?
    - Interaciont to next paint

- What does it measure about user interaction?
    - It measures how fast a response is towards a user interaction on the page.

- What does a poor score mean?
    - ≤ 200 milliseconds = ✅ Good (feels instant, responsive)
